works.

LoanJump

Alpha-build for a fun and stressful decision-based college simulator •Used GML to make game centered on series of timed binary choices •Choice system examined interaction of college-themed player attributes and evolved values over time

Space Trader

Multi-faceted economic simulation taking place in space •Created Java Swing GUI to simulate economic encounters with virtual traders, bandits and police •Game uses maps and inventory screens to navigate space regions and buy/trade/sell several goods

Memory- and sound-based game developed for LMC 2700 – Intro to Computational Media •Created full sound library and contributed onclick action scripts for complex memory tile game •Game uses sound and light to generate patterns for players to memorize and send to each other
